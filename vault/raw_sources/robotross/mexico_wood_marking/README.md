# Mexico Wood-Marking Logs

Place the Mexico wood-marking log files in this directory.

Recommended layout:
- keep the original log files directly in this folder if there are only a few
- if there are many files, create subfolders by day or machine, for example:
  - `2026-03-12/`
  - `2026-03-13/`
  - `machine-a/`

Guidelines:
- preserve original filenames when possible
- do not edit or reformat the raw files here
- if there is a note that explains machine, operator, or run context, add a separate `manifest.md` in this folder rather than modifying the logs

This folder is gitignored via `vault/raw_sources/`.
